# Home Settings

`coverPost` controls the large lead/cover entry on the index page.

You can point it at either a main post or a friend post:

- `the-discipline-of-paying-attention` — a bare slug means a main post.
- `post:the-discipline-of-paying-attention` — explicit main post.
- `friend:cyril-on-leaving-lagos` — a friend post (a file in `src/content/friends/`).

Use the filename slug without `.md`. The "Latest, all kinds" list stays
chronological across both main posts and friend posts, and does not depend on
this setting.
